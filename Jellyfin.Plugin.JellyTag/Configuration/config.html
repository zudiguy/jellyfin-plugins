JellyTag Configuration

Configure quality badges for your media posters. This plugin automatically parses .strm filenames to extract metadata.

General Settings

Master toggle to enable or disable all badge overlays
Where to display badges on the poster
Badge size percentage (50-200%). Default is 100%

Badge Categories

Select which types of badges to display:

Display resolution badges (4K, 1080p, 720p, etc.)
Display HDR format badges (DV, HDR10+, HDR10, HLG)
Display video codec badges (HEVC, H.264, AV1, etc.)
Display audio codec and channel badges (TrueHD Atmos 7.1, EAC3 5.1, etc.)
Display source badges (Remux, WEB-DL, BluRay, AMZN, etc.)

Filename Parsing

This plugin is designed for .strm files. It parses filenames to extract metadata.

info

Example Movie Format

The Amazing Spider-Man (2012) [Remux-2160p HEVC DV HDR10 10-bit TrueHD Atmos 7.1]-FraMeSToR.strm

Example TV Show Format
